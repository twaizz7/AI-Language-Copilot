# 🌍 AI Language Learning Assistant

一个基于大模型的语言学习工具，支持句子翻译、词性分析、语法结构解析，帮助用户高效学习英语。

---

## 🚀 项目亮点

- ✨ 基于大模型实现语法结构解析
- 🌐 提供 Web 界面（Streamlit）
- 🧠 支持逐词翻译 + 句法分析
- 🕘 支持历史记录功能（提升用户体验）

---

## 🛠 技术栈

- Python
- Streamlit（前端界面）
- 阿里云通义千问（DashScope API）
