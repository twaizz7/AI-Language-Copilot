# 🌍 AI Language Copilot

## 📌 项目简介
这是一个基于AI的语言学习工具，帮助用户学习韩语和英语。

---

## 🎯 项目背景
在学习语言时，很多翻译工具只能给出结果，但无法解释句子的结构和语法。

---

## 功能

- 英文句子逐词翻译
- 词性标注
- 句子结构分析（主谓宾）
- 语法解析
- 自然语言翻译

## 🧠 项目目标
构建一个AI工具，帮助用户理解句子结构，而不仅是翻译结果。
## 技术栈

- Python
- 通义千问（阿里云）
- DashScope API
- ## 运行方式

```bash
pip install -r requirements.txt
python main.py
